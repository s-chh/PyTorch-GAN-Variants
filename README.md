# PyTorch-GAN-Variants
PyTorch implementations of various Generative Adversarial Networks (GAN) variants:

 1. [Vanilla-GAN](https://github.com/s-chh/Pytorch-Vanilla-GAN)
 2. [Small GAN](https://github.com/s-chh/Pytorch-SmallGAN)
 3. [Deep Convolution GAN (DCGAN)](https://github.com/s-chh/Pytorch-DCGAN)
 4. [Deep Convolution GAN32 (DCGAN32)](https://github.com/s-chh/Pytorch-DCGAN32)
 5. [Least Squared GAN (LSGAN)](https://github.com/s-chh/Pytorch-LSGAN)
 6. [Least Squared GAN (LSGAN) - MNIST](https://github.com/s-chh/Pytorch-LSGAN-MNIST)
 7. [Conditional GAN (cGAN)](https://github.com/s-chh/Pytorch-cGAN-conditional-GAN)
 8. [CycleGAN - Digits](https://github.com/s-chh/Pytorch-CycleGAN-Digits)
 9. [Wasserstein GAN (WGAN)](https://github.com/s-chh/Pytorch-WGAN)
 10. [Improved Wasserstein GAN (WGAN-GP)](https://github.com/s-chh/Pytorch-WGANGP)
 11. [StarGAN - Digits](https://github.com/s-chh/Pytorch-StarGAN-Digits)
