# Pytorch-cGAN
Pytorch implementation of conditional GAN using DCGAN architecture for generating 32x32 images of SVHN, CIFAR10, MNIST, FashionMNIST, and USPS datasets.

<br>
Change the DB variable to change the dataset.

For using the saved model to generate images, set LOAD_MODEL to True and EPOCHS to 0.
